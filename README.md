# Trabajo Práctico Individual

Este repositorio contiene la resolución de una serie de ejercicios prácticos de Python desarrollados durante el curso.

## Estructura de los ejercicios

El repositorio está organizado en archivos individuales, cada uno correspondiente a un ejercicio temático:

- **`ejercicio-01.py`**: Ejercicio sobre condicionales e `input`. Implementa una validación para determinar si el usuario es mayor o menor de edad.
- **`ejercicio-02.py`**: Ejercicio sobre el bucle `while`. Programa que valida una contraseña de acceso.
- **`ejercicio-03.py`**: Ejercicio sobre bucle `for` y listas. Filtra números mayores a 10.
- **`ejercicio-04.py`**: Ejercicio sobre funciones. Implementa el cálculo del área de un rectángulo.
- **`ejercicio-05.py`**: Ejercicio sobre diccionarios y recorrido. Muestra el contenido de un diccionario mediante un bucle.
